# 开局训练营 ♞

国际象棋开局训练应用，帮助小朋友学习和练习各种开局变化。

## 快速开始

```bash
# 1. 安装依赖（只需要第一次）
npm install

# 2. 启动开发服务器
npm run dev
```

浏览器打开 http://localhost:5173 即可使用。

## 手机上使用

电脑和手机连同一个 WiFi，启动时用：

```bash
npm run dev -- --host
```

然后用手机浏览器访问终端显示的 Network 地址（类似 http://192.168.x.x:5173）。

## 部署上线

```bash
npm run build
```

生成的 `dist/` 文件夹可以部署到任何静态服务器：
- Vercel: `npx vercel dist`
- GitHub Pages
- Netlify
